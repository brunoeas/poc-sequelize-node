# poc-sequelize-node

Uma POC para testar a implementação da biblioteca Sequelize no NodeJS fazendo um CRUD simples

# Configs

Crie um arquivo chamado `config.json` na pasta `config` na raiz do projeto e coloque as configurações do banco:

```
{
  "development": {
    "username": "...",
    "password": "...",
    "database": "...",
    "host": "...",
    "port": "...",
    "dialect": "..."
  }
}
```
